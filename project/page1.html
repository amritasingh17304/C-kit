Hello World C program

#include < stdio.h>
int main()
{
printf("HELLO WORLD");
return 0;
}

Write the above code in any compiler and you will see the output on the screen as hello world.
So,Congratulations you have finally made your first C program

Let us learn what each and every line of the code meant

Firstly "#include < stdio.h> is actually used to include all the header files in the program.
Without this you won't get access to any of the features available in the C language."

Next is the int main() which is the place where every C file starts to execute.
Here int implies the return type of the function.
int is a datatype which we will learn further.

return 0 means the value returned after the function is called.
It denotes the success of the program

The printf statement is the same as the name implies.
It prints the value written inside it.
